MinimalApi

The goal of this project is to create a Minimal Api in .Net using MySql and testing it with MSTest.

This Api has functions to create, read, update and delete admins and vehicled.

The Admins has two roles: Adm and Editor:
- The Adm has all CRUD(create, read, update and delete) functions for Adins and Vehicle.
- The Editor only has the Create an Read vehicle functions.
